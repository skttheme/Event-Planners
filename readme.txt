== Description ==

SKT Event Planners is a celebration, marriage, conference, grand events, bookings, parties, happenings, outreach campaigns, competition, contest, tournament, game, fixture, meetings, encounter type of website template which is responsive and Gutenberg editor friendly. It comes with ready to import 63+ Elementor templates. On any occasion, business, incident or similar such conferences or parties with speakers, guests are organized. Can be used by community centers, event managers, wedding, engagement, naming ceremony, motivational coach. Compatible with WooCommerce, multilingual plugins, contact form plugins for call to action. Can accept donations via donation plugins, compatible with events calendar plugins as well as table plugins.
